Alesis DM10 MKII Studio Mesh Kit

11 ratings

Electronic Drum Set

  • Made available in Europe exclusively by Thomann
  • Successor to the Alesis DM10 studio mesh kit
  • The new DM10 MKII Advanced Electronic Drum module is based on the Alesis Crimson module
  • Contains 671 sounds
  • 54 Preset and 20 user kits
  • 120 Preset and 5 User songs
  • 15MB of memory for audio samples
  • Additional input for a fourth Tom (second crash input already used)
  • 2 x 6.3 mm mono-jack master outputs
  • 1 x 3.5 mm stereo headphone output jack
  • 1 x 3.5 mm stereo jack Aux-In
  • MIDI In
  • USB for MIDI connections and for loading new sounds
  • Including drum rack, connecting cable, drum sticks and tuning key

Set Configuration:

  • 1 x 10" Two-zone mesh head snare drum
  • 2 x 08" Two-zone mesh head tom pads
  • 1 x 10" Two-zone mesh head tom pad
  • 1 x 8" Mesh head kick pad
  • 1 x 12" Two-zone hi-hat pad
  • 2 x 12" Two-zone crash cymbal pad
  • 1 x 14" Three-zone ride cymbal pad
